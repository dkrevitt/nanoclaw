# Review Criteria

## Must Have

- Average 3,000+ views in last 20 videos (for video) OR 3,000+ followers with good engagement (for Twitter)
- Produces content for people building software (any skill level) OR using AI tools for business automation
- Has posted about:
  - Major AI coding agents (Cursor, Claude Code, Kilo Code, Lovable, etc) OR
  - Using AI models for coding OR
  - OpenClaw/ClawdBot/MoltBot (for ANY use case) OR
  - AI workflow automation tools
- Produces video content (YouTube, TikTok, Instagram) OR substantive text content (Twitter threads, technical posts)
- Creates content in English

## Nice to Have

- Profile mentions they are a developer or software engineer
- Posts video content in addition to text
- Posts about AI workflow automation or no-code/low-code tools

## Exclude If

- Never posts about AI coding agents, AI models for coding, OpenClaw/ClawdBot, or AI workflow automation

## Pre-filters

- minFollowers: 500
- requireEnglish: true
- maxDaysInactive: 60
- requireRecentPosts: true
