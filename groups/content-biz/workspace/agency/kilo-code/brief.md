# Kilo Code - Agency Client

**Website:** [kilocode.ai](https://kilocode.ai)
**Stage:** Growth
**Status:** Closed (worked 1 month)

## Company Overview

Kilo Code is an AI coding assistant. Direct competitor to Cursor, GitHub Copilot, and other AI coding tools.

## Program Goals

- Scale developer influencer program to 1M views/mo across platforms
- Drive awareness in two segments:
  1. **Community**: Individual developers
  2. **Teams**: Engineering managers + leaders
- Budget: Under $100k/mo when fully ramped (50/50 split between segments)

## Target Platforms

| Platform | Community Segment | Teams Segment |
|----------|-------------------|---------------|
| YouTube | Vibe coding channels (Tina Huang), daily AI shows (Matthew Berman) | Tech humor (Fireship, Programmers Are Also Human) |
| Newsletter | Daily AI newsletters (Nate B Jones) | Startup culture (Next Play, Hacker Newsletter), pre-AI SWE (Bytes.dev) |
| Podcast | - | SWE podcasts (Software Engineering Daily), startup (20VC, Lenny's) |

## Constraints

1. Build internal program - creator relationships owned by Kilo, handoff-ready
2. No intermediaries or marketplaces
3. Reduce ops overhead - repeat relationships, standardized compensation

## Timeline

- Month 1: Audit existing relationships, create creator database
- Month 2-3: Deploy budget to 2-3 new creators per segment/platform (10 new/month)
- Month 4+: Optimize, scale winning formats

## Pricing

- Initial: $15k/mo management fee up to $75k spend
- Scaled: 20% of spend beyond $75k/mo
- Term: 6 months initial

## What Happened

Worked with Kilo for 1 month. Program didn't continue.

## Learnings

- Teams segment requires iteration + format testing
- Engineering leadership audiences are harder to reach
- Need sufficient ramp time (1-2 month lead times with creators)
