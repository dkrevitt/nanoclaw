# Target Niches for Agency Prospecting

Industries and verticals where we want to find new agency clients.

## Tier 1 (Active Outreach)

### AI Developer Tools
Companies building tools for developers working with AI.

| Signal | Examples |
|--------|----------|
| "AI" in company description | Cursor, Codeium competitors |
| Recent AI funding | Check Crunchbase, TechCrunch |
| Developer docs for AI features | API references, SDK docs |

**Why good fit:** Hot market, lots of funding, clear developer audience, active creator ecosystem, differentiation matters.

**Subtypes:**
- AI coding assistants
- AI infrastructure (inference, training, deployment)
- AI APIs (LLM providers, specialized AI services)
- Agent frameworks and tools

### Developer Infrastructure
Backend, DevOps, and platform tools.

| Signal | Examples |
|--------|----------|
| Developer-first GTM | Inngest, Temporal |
| Technical documentation | Backend/infra focused |
| Open source component | GitHub presence |

**Why good fit:** Technical audience that follows creators, complex products benefit from explanation, strong newsletter/YouTube creator ecosystem.

**Subtypes:**
- Background jobs / workflow orchestration
- Databases and data infrastructure
- CI/CD and DevOps
- Observability and monitoring
- API management

### Developer Productivity
Tools that make developers more effective.

| Signal | Examples |
|--------|----------|
| Code-related product | CodeRabbit, Graphite |
| IDE integrations | VS Code, JetBrains |
| Developer workflow focus | Documentation, collaboration |

**Why good fit:** Universal developer audience, easy to demonstrate value, strong creator content potential.

## Tier 2 (Opportunistic)

### Security / Compliance
Developer-focused security tools.

| Signal | Examples |
|--------|----------|
| Application security | SAST, DAST tools |
| Compliance automation | Vanta competitors |
| Identity / auth | Auth0 competitors |

**Challenges:** Can be enterprise-focused (wrong motion), sensitive topics (harder content).

### Data Tools
Data engineering and analytics platforms.

| Signal | Examples |
|--------|----------|
| Data pipelines | Dagster, Prefect |
| Analytics platforms | Developer-focused |
| Data quality | Testing, monitoring |

**Challenges:** Can be very enterprise, smaller creator ecosystem than pure dev tools.

## Avoid

| Niche | Reason |
|-------|--------|
| Pure enterprise software | Wrong GTM motion, too slow |
| Consumer products | Different marketing motion entirely |
| Non-technical B2B | Can't find right creators |
| Commoditized tools | Low differentiation, price competition |
| Series C+ companies | Usually have internal resources |

## Discovery Tactics

### News Monitoring (use Tavily)
- Recent Series A/B funding in target niches
- Companies launching new developer products
- Competitors starting creator programs

### Community Signals
- Active on Hacker News, Reddit programming subs
- Product Hunt launches
- Developer conference sponsors

### Hiring Signals
- Hiring Head of Marketing or DevRel
- Building out marketing team
- Job posts mentioning "creator" or "influencer"
