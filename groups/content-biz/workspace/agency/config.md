# Agency Line Configuration

## Overview

We help B2B tech companies build and manage developer/technical creator programs. Our value is in creator relationships, content strategy, and program operations - not just media buying.

## Ideal Client Profile

| Attribute | Target |
|-----------|--------|
| Stage | Series Seed - Series B |
| Budget | $50k+/mo creator spend (minimum) |
| Buyer | Head of Marketing (or DevRel lead) |
| Product | Developer tools or adjacent technical/semi-technical B2B products |
| Audience | Sophisticated/technical decision makers |

## Service Tiers

### Phase 1: Market Mapping ($15k flat, 3 weeks)

Discovery project for companies who need clarity before scaling spend.

**Deliverables:**
- Persona definitions + topic cluster map
- Creator database (50+ creators)
- Budget + ROI model with recommendations

### Phase 2: Program Management ($10-15k/mo retainer)

Ongoing creator program management.

**Deliverables:**
- Weekly: Budget deployment, performance metrics, creator pipeline
- Quarterly: Strategic review, audience insights, optimization
- Ongoing: Creator agreements, content briefs, weekly creator digest

**Requirements:**
- $50k/mo minimum creator spend
- 6-month minimum commitment

## Our Role vs Client Role

**We do:**
- Source creators (direct relationships, no intermediaries)
- Write content briefs
- Review drafts for accuracy + messaging alignment
- Coordinate revisions with creators
- Manage creator payments (via Ramp virtual card)
- Weekly briefing docs to creator roster

**Client does:**
- Provide product access, demos, educational resources
- Final approval on content (goal: establish criteria to reduce review burden)
- "How did you hear about us?" tracking where possible

## Default Criteria

### must_have (defaults)
- Audience matches client's target customer
- Has done sponsored content before (or willing to start)
- Professional content quality
- Engagement rate above 2%

### exclude_if (defaults)
- Has worked with client's direct competitors recently
- Pricing above client budget
- Fake/bot engagement patterns

## Measurement Framework

- **Primary:** Views at target CPM (~$100)
- **Secondary:** Activated signups (shortlinks, promo codes, "how did you hear")
- **Reporting:** Weekly deployment + performance, monthly strategic review

## Key Learnings

- Newsletters are fastest to activate and have best attribution
- YouTube requires patience (1-2 month lead times)
- Repeat relationships with creators deliver better results than one-offs
- Standardized creator compensation reduces ops overhead
- "How did you hear about us?" is critical for attribution
- Direct creator relationships (no intermediaries) = better handoff potential
