# Sponsor Criteria: Decoding AI

## Target Sponsor Categories

Paul's audience is AI engineers building production systems. Best-fit sponsors are in:

### Priority Targets (New Focus)

1. **Audio for AI**
   - Speech-to-text / text-to-speech APIs
   - Voice AI platforms
   - Audio processing tools
   - Examples: Deepgram, AssemblyAI, ElevenLabs, Play.ht

2. **On-device AI**
   - Edge inference platforms
   - Model optimization/quantization tools
   - On-device ML frameworks
   - Examples: Qualcomm AI, Apple MLX ecosystem, ONNX Runtime

### Proven Categories

3. **MLOps / LLMOps**
   - Experiment tracking (Comet, Weights & Biases)
   - Model deployment (Modal, Replicate)
   - Pipeline orchestration (Prefect, Dagster)
   - Feature stores (Hopsworks, Tecton)

4. **Vector Databases / Search**
   - Qdrant, Pinecone, Weaviate, Chroma
   - Hybrid search (MongoDB Atlas Vector Search)

5. **Inference / Compute**
   - GPU providers (Groq, Together, Fireworks)
   - Inference optimization (Unsloth, vLLM)

6. **Data Infrastructure**
   - Databases (MongoDB, Snowflake)
   - Data platforms

## Pre-Filters

minFunding: Seed+ (but prefer Series A+)
requireB2B: true
requireAIFocus: true

## must_have

- **AI/ML product**: Must be relevant to AI engineers building production systems
- **Technical depth**: Product should have interesting architecture to explain
- **Budget**: $5k+ for standalone, $40k+ for series
- **Collaborative**: Willing to work on technical accuracy

## nice_to_have

- Open-source component (Paul's audience loves OSS)
- Python SDK (primary language of audience)
- Production use cases to highlight
- Interesting technical approach worth explaining

## exclude_if

- Consumer AI products (not Paul's audience)
- Hype-focused / no real product
- Requires heavy marketing language (Paul's style is technical)
- Competes with active sponsors

## Sponsorship Packages

| Package | Price | Deliverables |
|---------|-------|--------------|
| Standalone LinkedIn | $2,000 | 1 LinkedIn post |
| Standalone Newsletter | $5,000 | 1 newsletter issue |
| Reddit/HN Roundup | $2,500 | Summary post with HN cross-post |
| Book Series | $40,000 | 10 LinkedIn + 2 Newsletter + 4 Roundups |

## Outreach Approach

Paul's sponsorships work best when the sponsor is featured naturally in educational content. Pitch should emphasize:

1. How their product fits into production AI architecture
2. What technical tutorial could feature their product
3. Why their product is interesting to explain (not just advertise)
