## What It Is

System Design breaks down how large-scale systems work using real-world examples from companies like Stripe, Uber, WhatsApp, PayPal, Google, and Meta. Focuses on core engineering problems: scalability, reliability, distributed systems, APIs, and data architecture.

**Positioning:** Readers come to understand how real systems behave in production, not for surface-level trends.

## Content Types

1. **System design breakdowns** - How X company built Y system
2. **Interview prep** - System design interview patterns
3. **Architecture deep dives** - Specific technical patterns (stacked diffs, context engineering)
4. **Best practices** - Code reviews, Git commands, engineering team patterns
