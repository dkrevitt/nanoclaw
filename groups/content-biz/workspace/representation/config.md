# Representation Line Configuration

## Overview

We represent creators for sponsorship sales. Our focus is on technical creators (AI/ML, software engineering, system design) who produce educational content.

**Current roster:**
- Decoding AI (Paul Iusztin) - AI engineering, 90k+ audience
- System Design Newsletter (Neo Kim) - Backend/infra, 221k+ subs
- Deep Learning w/ Yacine - Deep learning research, 37k subs

**Former:**
- Kai Lentit / Programmers Are Also Human - Tech comedy (instructive example of ideal creator)

## Value Proposition

We handle sponsor outreach so creators can focus on content. Our approach:
- Long-term sponsor relationships (series > one-offs)
- Strategic sponsor matching (audience fit, not just budget)
- Content integration (not just banner ads)
- Full-service: outreach → negotiation → contracting → coordination

## Commission Structure

Typical: 15-20% of sponsor revenue generated

## Ideal Creator Profile

| Attribute | Target |
|-----------|--------|
| Audience | 10k+ (quality over quantity) |
| Niche | Technical/educational (AI, engineering, dev tools) |
| Platforms | Newsletter, YouTube, LinkedIn, X |
| Content | Production-focused, practical, educational |
| Track record | Consistent posting, engaged audience |

## Default Pre-Filters (for prospecting)

```
minFollowers: 10000
maxDaysInactive: 14
requireEnglish: true
```

## Default Platforms

Search order:
1. YouTube
2. Twitter/X
3. LinkedIn
4. Newsletter (Substack, Beehiiv)

## Standard Criteria

### must_have (defaults)
- Growing audience (check recent trends)
- Consistent posting schedule (weekly minimum)
- Technical/educational content
- Open to representation

### exclude_if (defaults)
- Already represented exclusively
- Inconsistent posting
- Primarily promotional/affiliate content
- Controversial content history

## Sponsor Discovery Approach

When finding sponsors for represented creators:
1. Identify sponsor categories that match creator's audience
2. Search for brands sponsoring similar creators (check "#ad" posts)
3. Use Apollo for company enrichment + contact finding
4. Prioritize brands with existing creator programs

## What Makes a Great Sponsor Match

- **Audience alignment**: Sponsor's ICP matches creator's audience
- **Content fit**: Sponsor's product can be featured naturally in educational content
- **Budget**: $5k+ per placement (or series commitment)
- **Collaboration**: Willing to work on technical accuracy

## Notes

- Humor + technical depth is rare and valuable (Kai Lentit example)
- Series sponsorships outperform one-offs
- LinkedIn posts often reach wider audience than newsletters for B2B
- Architecture diagrams drive engagement for technical creators
