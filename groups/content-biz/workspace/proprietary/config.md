# Proprietary Line Configuration

## Overview

Publications we own and operate. Currently:
- **Technically** (~72k subs) - Tech explainers for startup operators

## Discovery Goals

- **Contributors**: Find writers, creators, correspondents for publications
- **Sponsors**: Find brands that want to sponsor our publications

## Current Priority: Technically

**Active search:** "The Analyst" correspondent
- Opinionated, funny, analytically minded
- 20hrs/week, Americas timezones
- See `technically/contributor-criteria.md`

**Sponsor shift:** Moving from dev tools to startup operators (founders, product, finance/ops, sales+marketing)

## Default Pre-Filters (for contributor search)

```
minFollowers: 0 (talent over audience for contributors)
maxDaysInactive: 30
requireEnglish: true
requireTimezone: Americas
```

## Default Platforms

Search order for contributor discovery:
1. X/Twitter (find voices)
2. LinkedIn (find backgrounds)
3. Substack (find writers)
4. YouTube (find on-camera presence)

## Standard Criteria

### must_have (defaults)
- Creates original content in publication niche
- Posts consistently (at least 2x/month)
- Content quality matches publication standards
- Original takes (not just aggregation)

### exclude_if (defaults)
- Primarily promotional/affiliate content
- Topic mismatch with publication
- No original work or takes
- Full-time employment (for correspondent roles)

## Sponsor Discovery Approach

For finding sponsors for proprietary publications:
1. Identify categories matching audience shift (startup operators, not just devs)
2. Search for brands marketing to founders, product, finance, GTM leaders
3. Use Apollo for company enrichment + contact finding
4. Check recent funding (budget signals)
